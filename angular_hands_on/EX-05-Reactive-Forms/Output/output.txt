Project Name: student-course-portal-5
Framework: Angular 17+ (Standalone Architecture)

1. Reactive Form Structure (FormBuilder):
   - studentName: [Validators.required, Validators.minLength(3)]
   - studentEmail: [Validators.required, Validators.email], asyncValidator: simulateEmailCheck
   - courseId: [Validators.required, noCourseCode]
   - preferredSemester: ['Odd', Validators.required]
   - agreeToTerms: [false, Validators.requiredTrue]
   - additionalCourses: FormArray for dynamic course input controls

2. Custom Validators Implemented:
   - Sync Validator `noCourseCode`: Rejects course IDs starting with 'XX'
   - Async Validator `simulateEmailCheck`: Simulates backend check returning { emailTaken: true } for addresses containing 'test@'

3. Dynamic Form Controls (FormArray):
   - `additionalCourses` getter providing typed access to FormArray
   - `addCourse()` pushes new FormControl with required validation
   - `removeCourse(index)` removes dynamic control at specified index

4. Value Access & Submission Logic:
   - Difference between `.value` (enabled controls) vs `.getRawValue()` (includes disabled controls)
   - Recursively marks all controls as touched (`markFormGroupTouched`) on invalid submission attempt.
