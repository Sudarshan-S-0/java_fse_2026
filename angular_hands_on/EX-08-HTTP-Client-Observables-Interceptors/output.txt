Project Name: student-course-portal (EX-08 API Integration)
Backend Server: json-server (http://localhost:3000)

1. HttpClient Provider Setup:
   - Configured `provideHttpClient(withInterceptors([authInterceptor, loggingInterceptor]))` in app.config.ts

2. RESTful API Operations Implemented:
   - GET /courses -> Fetch all courses (`http.get<Course[]>('http://localhost:3000/courses')`)
   - GET /courses/:id -> Fetch single course details by ID
   - POST /courses -> Create new course record
   - PUT /courses/:id -> Update existing course record
   - DELETE /courses/:id -> Remove course record

3. RxJS Operators & Error Handling:
   - `pipe(map(...), tap(...), catchError(...), retry(2))`
   - Centralized HttpErrorInterceptor catching 401, 404, 500 status codes with friendly notifications.

4. HTTP Interceptors Architecture:
   - AuthInterceptor: Appends `Authorization: Bearer <token>` to outgoing request headers.
   - LoggingInterceptor: Measures and logs request-response latency to browser console.
